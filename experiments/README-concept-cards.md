# Web3 概念卡片 — 交互式学习工具

## 🧠 概述

基于 [AI × Web3 School Handbook](https://aiweb3.school/zh/handbook/) 的 **网络（Network）**、**密码学（Cryptography）**、**钱包（Wallet）** 三章内容，生成的一个交互式概念卡片网页（类似 Quizlet）。

## 🤖 AI 做了什么

- 从 Handbook 三章中提取了 **16 个核心概念**（区块、共识、PoS、测试网、L2、Rollup、哈希、公钥、私钥、签名、Merkle Tree、EOA、助记词、交易、Gas、Explorer）
- 为每个概念编写了**定义摘要**和**最小实践/安全提示**
- 生成了完整的 HTML + CSS + JavaScript 交互页面：
  - 点击翻转卡片
  - 按主题分类筛选（全部 / 网络 / 密码学 / 钱包）
  - 标记已掌握 / 进度追踪
  - 随机排序
  - 键盘快捷键（← → 翻页，空格翻转，M 标记掌握）

## ✋ 我人工做了什么

- **确认内容准确性** — 逐条对比 Handbook 原文，修正了 AI 对 Merkle Tree、Rollup 等概念的不完整描述
- **补充安全提醒** — 私钥管理规则、助记词风险提示等 AI 概括不够强的内容，人工强化了语气
- **调整交互设计** — 卡片布局、配色方案、进度条和按钮位置按学习场景做了修改
- **修复了示例格式** — AI 生成的 example 字段部分缺少实际场景，人工补充了具体用例

## ⚠️ 不可靠输出记录

| 问题 | AI 输出 | 人工修正 |
|------|---------|---------|
| EOA 定义 | 说过简 | 补充了自动化能力弱的限制 |
| PoS 描述 | 忽略 Slashing | 补充了惩罚机制 |
| Transaction 状态 | 只写了成功/失败 | 补充了 5 种状态流程 |
| 推荐体验 | 没有针对 Web3 新手优化卡片难度分层 | 全部概念标注了难度层级（初级/中级/高级） |
